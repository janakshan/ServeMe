## Education Service

The Education Service is a comprehensive learning platform designed for students from Primary level through A/L examination preparation.

### Home Dashboard

**Personal Learning Hub**
- **Progress Tracking**: Monitor your learning journey with detailed statistics
  - Current study streak counter
  - Total XP (experience points) earned
  - Completed courses count
  - Current learning level (Beginner, Intermediate, Advanced)
- **Weekly Goals**: Set and track weekly learning objectives
  - Progress indicator showing completed vs. target sessions
  - Visual progress bars
- **Quick Stats Overview**:
  - Last course you studied with progress percentage
  - Next lesson recommendations
  - Upcoming class reminders
- **Achievement System**:
  - View recent achievements and milestones
  - Badges for consistency, mastery, and top performance
  - Motivational icons and descriptions

**Quick Access**
- Direct shortcuts to your most-used features
- One-tap navigation to courses, live classes, and exams
- Personalized recommendations based on your learning history

### Courses

**Multi-Level Education System**
The platform offers courses across four educational levels:
1. **Primary Education** (Grades 1-5)
2. **Junior Secondary** (Grades 6-8)
3. **O/L (Ordinary Level) Preparation**
4. **A/L (Advanced Level) Preparation**

**Course Features**
- **Comprehensive Course Catalog**:
  - Browse courses by category (Primary, Junior Secondary, O/L, A/L)
  - Filter by subject (Mathematics, Science, Tamil Language, English, History)
  - Filter by difficulty level (Beginner, Intermediate, Advanced, Expert)
  - Search functionality to find specific courses or instructors

- **Course Details**:
  - Complete course descriptions
  - Duration information (terms/years)
  - Instructor profiles with qualifications
  - Student ratings and reviews
  - Total enrolled students count
  - Pricing information

- **Enrollment System**:
  - One-click enrollment for available courses
  - Progress tracking for enrolled courses
  - Visual progress bars showing completion percentage
  - Resume learning from where you left off

- **Subjects Available**:
  - Mathematics (Basic arithmetic to Advanced Calculus)
  - Science (Integrated Physics, Chemistry, Biology)
  - Languages (Tamil, English, Sinhala)
  - Social Studies (History, Geography, Civics)

### Live Classes

**Interactive Learning Sessions**
Connect with expert teachers in real-time through our live class platform.

**Class Management**
- **Multiple Views**:
  - All Classes: Browse all available live sessions
  - My Classes: View only classes you're registered for

- **Class Status Tracking**:
  - **Live Now**: Currently ongoing classes with "Join Now" button
  - **Upcoming**: Future scheduled classes with registration
  - **Completed**: Past classes with recording access

**Class Information**
- Detailed class schedules with date and time
- Class duration information
- Instructor details and qualifications
- Subject and topic coverage
- Current enrollment vs. maximum capacity
- Visual capacity indicators

**Registration System**
- Easy registration for upcoming classes
- Registration status tracking
- Automatic reminders for registered classes
- Bookmark feature for classes you're interested in

**Advanced Calendar Integration**
- **Multiple Calendar Views**:
  - **Day View**: Hour-by-hour schedule for selected date
    - Time slots from 7 AM to 10 PM
    - Visual representation of class timings
    - Quick class details in each time slot
  - **Week View**: 7-day overview with classes
    - Navigate forward/backward by week
    - See all classes across the week at a glance
    - Quick day selection to view details
  - **Month View**: Full monthly calendar
    - Visual indicators for classes on each day
    - Color-coded by status (Live, Upcoming, Completed)
    - Select any date to view that day's classes

- **Smart Navigation**:
  - Month and year picker for quick date selection
  - Today button to return to current date
  - Smooth transitions between views
  - Synchronized date selection across all views

- **Visual Indicators**:
  - Red dots for live classes
  - Purple dots for upcoming classes
  - Green dots for completed classes
  - Multiple indicators if a day has various class types

**Class Recordings**
- Access recordings of completed classes
- Watch at your own pace
- Review difficult concepts
- Perfect for revision before exams
